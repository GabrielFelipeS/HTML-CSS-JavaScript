HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é mais recente evolução de padrão que define a linguagem HTML

Esta versão traz importante mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade

posiblita o uos de novos recusos possiveis apenas com a aplicação com a aplicaçao de outras tecnologias
Muitos recuros do HTML 5 têm sido contruídos com a intenção de serem executados em dispositivos móveis como smartphones e tablets

Historia

Entre 1991 e 1999, o HTML se desenvolveu a parir do versão 1 até o versão 4. Em 2008, a primeira versão publicada de HTML 5 foi lançada

As primeiras versões de HTML foram definidas com regras sintáticas flexiveis, o que ajudou aqules sem familiariade com a Web, Com o tempo aumentou a tendência em tornar cada vez mais rigida. Apesar disso, por questões históricas(retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web desenvolivdas com as primeiras versões de HTML

Algumas curiosidades sobre a Linguagem HTML:

  1. HTML não é uma linguagem de programação e sim uma linguagem de marcação
  2. Foi inventada na década de 1990 por um cientista chamado Tim Berners-Lee
  3. É possiveis criar um site apenas com bloco de notas
  4. Organizada o código, usando indentação, ajuda na manutenção
